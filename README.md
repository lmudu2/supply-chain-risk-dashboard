# Supply Chain Risk Dashboard

A comprehensive AI-powered dashboard for monitoring, analyzing, and predicting supply chain risks. This application leverages machine learning to predict shipment delays and provides insights into Geopolitical, Financial, Operational, and Quality risks across a global supply network.

## 📖 Overview

Supply Chain Risk Analytics is built with **Streamlit** and uses a **Random Forest Classifier** to analyze likelihood of shipment delays. It simulates a global dataset of suppliers, products, and logistics routes to demonstrate how various risk factors intersect to impact supply chain resilience.

The dashboard allows supply chain managers to:
-   Visualize risk concentration across regions and suppliers.
-   Predict probability of delays using granular risk metrics.
-   Simulate "What-If" scenarios to see how changing suppliers or routes affects risk.
-   Monitor live metrics on Quality, ESG, and Financial stability.

## ✨ Key Features

-   **Interactive Filtering:** Drill down by Region (e.g., US, EU, Asia), Country, Supplier, Product, and Shipment Mode.
-   **AI Risk Prediction:** Real-time calculation of delay probability based on:
    -   *Geopolitical Instability* (Country Risk Index)
    -   *Climate Risks* (Typhoon belts, Flood plains)
    -   *Operational Metrics* (Capacity Utilization, Port Congestion)
    -   *Supplier Health* (Credit Ratings, Relationship Scores)
-   **Granular Analysis Tabs:**
    -   **🌍 External Risk:** Heatmaps of geopolitical and climate threats.
    -   **💰 Financial:** Supplier credit ratings, cost competitiveness, and bankruptcy risk.
    -   **⚙️ Ops Performance:** Late shipment tracking, capacity utilization, and logistics delays.
    -   **✅ Quality & ESG:** Defect rates, audit statuses, and sustainability scores.
    -   **🤝 Relationship:** Collaboration scores and innovation indices.
    -   **🤖 Risk Simulator:** Interactive tools to test how changes in parameters impacts the overall risk profile.

## 🚀 Getting Started

### Prerequisites

-   Python 3.8 or higher installed on your machine.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/lmudu2/supply-chain-risk-dashboard.git
    cd supply-chain-risk-dashboard
    ```

2.  **Create a virtual environment (Optional but Recommended):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

### Running the App

Execute the following command in your terminal:

```bash
streamlit run app.py
```

The application will launch automatically in your default web browser at `http://localhost:8501`.

## 📂 Project Structure

-   `app.py`: The main entry point for the Streamlit application. Contains all UI logic, data generation, and model training.
-   `requirements.txt`: List of Python packages required to run the project.
-   `SupplierRiskScorev2.ipynb`: Jupyter Notebook likely used for initial data exploration and model prototyping.
-   `supply_chain_risk_analysis.csv`: (Generated/Included) Dataset used or generated by the application.
